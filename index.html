Cloud Computing Assignment 3: AWS Hosting Solution

Introduction

This static webpage provides an overview of the cloud solution for the student records web application, hosted on AWS as part of the Cloud Computing Assignment 3. The web application is designed to be highly available, scalable, load balanced, and secure.

Architecture Overview

The architecture consists of the following components:

  • EC2 Instances to host the web application
  • Amazon RDS for the student records database
  • Elastic Load Balancer to balance the traffic
  • Auto Scaling for scalability during peak load times
  • Security Groups to control access

Conclusion

This solution demonstrates the use of AWS services like EC2, RDS, and Load Balancing to host a web application in a secure and scalable manner. The static webpage itself is hosted using Amazon S3.
