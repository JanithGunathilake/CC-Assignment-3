Cloud Computing Assignment 3 - Student Records Web Application

Introduction

This static web page provides a detailed overview of the cloud-based solution created using Azure for hosting a student records web application. The architecture was designed to meet the requirements of availability, scalability, security, and performance.

Azure Configurations

1. Azure App Service

Azure App Service is used to host the web application, providing auto-scaling and high availability.

2. Azure SQL Database

The student records are stored in an Azure SQL Database, which is securely hosted within a private subnet.

3. Load Balancer

An Azure Load Balancer is used to distribute incoming traffic across multiple instances of the web application.

4. Security Configurations

Network Security Groups (NSG) were used to secure access to the database and web servers.

Conclusion

The Azure-based solution was successfully implemented, ensuring that the student records web application is highly available, secure, and scalable. This POC demonstrates how cloud infrastructure can enhance the performance and reliability of web applications.
